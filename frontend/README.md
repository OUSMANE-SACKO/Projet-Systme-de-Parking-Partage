# ParkingApp Frontend
[empty]
Interface ultra épurée et futuriste pour ParkingApp.

## Lancer le front localement

1. Compiler le SCSS en CSS :
   ```sh
   sass styles.scss styles.css
   ```
   (ou utiliser un plugin VS Code SCSS/SASS)

2. Ouvre `index.html` dans ton navigateur.

## Fonctionnalités
- Design glassmorphism, néon, dark mode
- Formulaires animés (connexion/inscription)
- Responsive mobile
- Prêt à brancher sur l’API backend

## Personnalisation
- Modifie `styles.scss` pour changer les couleurs, effets, etc.
- Ajoute tes appels API dans `app.js`.
